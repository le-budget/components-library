# BudgetNavbarMenuItem

Item de menu pour `BudgetNavbarMenu`.

## Usage

```vue
<BudgetNavbarMenuItem active>
  <a href="#" class="w-full">Transactions</a>
</BudgetNavbarMenuItem>
```

## Props

- `active`: applique le style actif
- `disabled`: desactive l'item (visuel + interactions)

## Comportement responsive

- Mobile: un item par ligne
- Desktop: largeur automatique
