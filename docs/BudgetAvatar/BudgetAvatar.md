# BudgetAvatar

Avatar pour afficher une image utilisateur.

## Usage

```vue
<BudgetAvatar image="/avatars/user-01.jpg" />
```

## Props

- `image`: URL de l'image
- `shape`: `circle | rounded | square`
- `size`: `sm | md | lg | xl | xxl`
- `thickness`: `sm | md | lg`
- `color`: `primary | secondary | ghost | primary-success | secondary-success | primary-warning | secondary-warning | primary-error | secondary-error`
- `alt`: texte alternatif de l'image

## Tailles

- `sm`: ~48px
- `md`: ~64px
- `lg`: ~92px
- `xl`: ~128px
- `xxl`: ~256px
