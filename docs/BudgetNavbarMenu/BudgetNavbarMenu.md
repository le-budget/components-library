# BudgetNavbarMenu

Conteneur de navigation pour `BudgetNavbar`.

## Usage

```vue
<BudgetNavbarMenu>
  <BudgetNavbarMenuItem>
    <a href="#" class="w-full">Tableau de bord</a>
  </BudgetNavbarMenuItem>
</BudgetNavbarMenu>
```

## Props

- `ariaLabel`: libelle accessible de la liste (par defaut `Menu principal`)
- `align`: `left | center | right` (par defaut `left`)

## Comportement responsive

- Mobile: affichage vertical
- Desktop: affichage horizontal
