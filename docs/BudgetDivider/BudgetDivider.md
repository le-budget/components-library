# BudgetDivider

Separateur visuel horizontal ou vertical avec variantes de couleur alignees sur `BudgetRadio`.

## Usage

```vue
<BudgetDivider />
```

## Props

- `orientation`: `horizontal | vertical` (par defaut `horizontal`)
- `color`: `primary | neutral | success | warning | error` (par defaut `primary`)
- `lineStyle`: `solid | dashed | dotted` (par defaut `solid`)
- `thickness`: `sm | md | lg` (par defaut `md`)
