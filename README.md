# Components Library

Bibliotheque de composants Vue 3 pour le SaaS Budget.

## Prerequis

- Node.js 20+
- Yarn 4 (Corepack recommande)

## Installation

```bash
corepack enable
yarn install
```

## Scripts utiles

```bash
yarn dev
yarn build
yarn test
yarn storybook
yarn build-storybook
```

## Publication

La publication se fait sur GitHub Packages via un tag `X.Y.Z`.
